  • didValidate(did: string): boolean
    • Defined in Utils.ts:71

    Validates a DID. The following validation has to be met:

    1. The string must start with did:.
    2. The operation name must be all a-z 0-9 chars.
    3. The id part must be a 64 char hex number.
